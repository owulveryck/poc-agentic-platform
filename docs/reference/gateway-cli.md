Per [ADR-130](../decisions/ADR-130-gateway-naming.md), "gateway" now names
a *control point* (one distributed enforcement point), and the central
policy evaluator `cmd/ppg` is the **validation server**. This reference
moved to [validation-server-cli.md](validation-server-cli.md); this stub
remains because published material links to the old path.
